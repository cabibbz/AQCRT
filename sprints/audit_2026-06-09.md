# System Audit — 2026-06-09 (between Sprints 136 and 137)

**What:** 48-agent deep audit of the entire research system across 7 dimensions (core numerical
libraries, recent experiments S134-136, data integrity, documentation, loop infrastructure,
scientific claim chain, repo hygiene). Every medium+ finding was adversarially verified by 1-2
independent agents instructed to REFUTE it (high/critical: refute + reproduce lenses, unanimous
vote required). Outcome: **29 confirmed, 2 refuted, 22 low/unverified.** All confirmed findings
were fixed the same day. This file is the permanent record; CHANGELOG has the summary.

## Confirmed and FIXED — code

1. **gpu_utils.eigsh ordering trap (HIGH).** With `return_eigenvectors=False`, scipy returns
   eigenvalues DESCENDING, CuPy ASCENDING — so a naive `vals[1]-vals[0]` gap series would
   silently flip semantics crossing the dim>50k backend switch (q=4 at n=7→8). Reproduced
   exactly by two verifiers (CPU `[-9.0565×3, -9.1423]` vs GPU `[-9.1423, -9.0565×3]` at q=4
   n=8). No existing number was wrong (all legacy callers sort or use k=1). FIX: both branches
   now return ascending; docstring states it.
2. **fss_utils.fit_power_law r² broken with error bars (MEDIUM).** Weighted ss_res over
   unweighted ss_tot ⇒ r²=−334 on an excellent fit with small errors; alpha_err also ignored
   the absolute error scale (scale_covar default). Latent (no current caller passes errors).
   FIX: unweighted-residual r²; scale_covar=False when errors supplied.
3. **exp_136b double diagonalization (MEDIUM).** `gap_curve` called `charge0_gap` twice per
   g-point — 2x GPU cost of every production EP scan. FIX: single call.
4. **exp_136b silent edge-clamp (MEDIUM).** The sparse path clamped a window-edge argmin and
   fitted a parabola around a non-minimum without any flag. FIX: RuntimeError on edge hits
   (coarse unsafe path, and fine grid both).
5. **collapse_utils.scan_curve half-convention (MEDIUM).** Returned exactly half the declared
   canonical chi_F (forward-diff, no factor 2) while chif_utils declares ONE convention. All
   S130 exponents/collapses unaffected (global prefactor). FIX: canonical factor-2 + provenance
   note (raw S130 JSON curve values are old half-scale).
6. **format_stream.py crash path (HIGH).** Valid-JSON non-dict lines (e.g. a bare string)
   crashed the exception handler itself (`ev.get` on a str), killing the live view pipeline
   mid-sprint. FIX: isinstance guard + hardened handler.
7. **db_utils.query oldest-wins (MEDIUM).** `ORDER BY sprint DESC` + the standard
   `{r[4]: r[6] for r in rows}` consumer pattern silently kept the OLDEST sprint's value on
   cross-sprint duplicates. FIX: `sprint ASC` (newest-wins for dict consumers); verified no
   consumer relied on DESC.

## Confirmed — science (claims HEDGED in KNOWLEDGE/STATE; data untouched)

18. **"Two INDEPENDENT observables agree" (S136) was near-tautological (HIGH).** The thermal-gap
    EP and on-peak chi_F are dominated by the SAME single epsilon multiplet; the 2-level identity
    chi_F^peak × Im(g_EP)² ≈ const holds to ~2% across q and n. Agreement was expected by
    construction. REFRAME: the EP is a second ESTIMATOR of the same low-energy physics; the
    product-constancy itself is the clean quantitative result (single-multiplet dominance).
    A genuinely independent confirmation needs a non-spectral observable (order parameter /
    entanglement nu).
19. **"Marginal-operator LOG detected" (S135) is conditional (HIGH).** The no-log alternative
    (K=1.70 + surface term) fits n≤24 equally well; s is parametrization-unstable (−1.3..−1.8).
    The robust datum is the q4-vs-q3-control deficit CONTRAST (0.30 vs 0.00). Also: production
    DMRG ran at the chi=48 cap for ALL sizes with convergence verified only at n=12 —
    chi-doubling at n=24 is an open rigor item (queued Sprint 137).
20. **"nu(q=4)=2/3 CONFIRMED" overstated (MEDIUM).** One calibrated estimator, no error bar,
    undemonstrated log-insensitivity. Hedged to "1.45-1.49 at n≤10, consistent with exact 2/3,
    excludes nu≥5/6; the 2.0 asymptote ultimately rests on the exact literature nu".
21. **q=4 EP "climbs exactly as a marginal operator demands" (MEDIUM).** 4-6 sizes cannot
    distinguish s/lnL from generic L^-omega drift. The controlled statement: undershoot at
    matched n is 8.7% (q4) vs 3.9%/0.9% (q2/q3 controls) — 2-9x larger, consistent with but
    not uniquely demonstrating the log. Falsifier recorded: at DMRG sizes the slope must follow
    1.5−c/lnL (pass ~1.44 by L~30), not saturate near 1.40.
22. **Planned q=5,6 walking-crossover hunt was UNREACHABLE (MEDIUM).** The only recorded xi
    anchor ("xi>12", an n≤12 lower bound) hid the exact classical values (Buffenoir-Wallon
    1993; reproduced numerically to 4 digits by the verifier): xi_d = 2512 (q5), 159 (q6),
    48 (q7), 24 (q8), 10.6 (q10). DMRG n=20-40 brackets L~xi only for q≥8. Sprint 137
    RETARGETED to q=8-10 with q=5/6 as deep-shadow controls.
23. **"Walking = complex-CFT shadow" stated as identity (MEDIUM).** Data equally consistent
    with generic weak-first-order drift; q=7 rests on 3 slope pairs with a 4-9% method
    undershoot. Verifier's PRELIMINARY Coulomb-gas continuation (validated on exact q≤4):
    Re(1/nu_complex) = 1.534/1.563/1.588 for q=5/6/7 vs measured 1.51/1.65/1.78 — q=5 matches,
    q=6,7 rise PAST it toward the first-order value d=2. Proper Jacobsen-Wiese comparison
    queued before any further shadow language.
